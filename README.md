# 🌸 Iris Flower Classifier

A beginner-friendly machine learning project using Scikit-learn to classify iris flowers into Setosa, Versicolor, or Virginica based on petal and sepal features.

## 📚 Dataset
- Built-in Iris dataset from Scikit-learn
- 150 samples, 4 features, 3 classes

## 🔧 Tools Used
- Python
- Scikit-learn
- Pandas, Seaborn, Matplotlib
- Google Colab

## 📊 Features Used
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## 📈 Model Used
- Support Vector Classifier (SVC) with linear kernel

## ✅ Accuracy
- Achieved over **95% accuracy** on the test set

## 🔍 How to Run
- Open the notebook in Google Colab
- Run all cells to train and test the model
